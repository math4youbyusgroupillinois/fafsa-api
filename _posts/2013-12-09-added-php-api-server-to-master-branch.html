For this project, I'm using the gh-pages branch as a place to put anything that is part of communicating and interacting with the project, while I'm putting all code, data, etc under the master branch in this repository.

I just added the server side code for the PHP implementation of the FAFSA API server. I'm using Slim Framework to handle all the server side methods for the API. You can access all of the code under the /master Github repository.

Upon request I'm happy to work on Python, Ruby or Node.js versions of the server as well to support any efforts--for now I will just work with the PHP versions. Let me know if you are interested.
