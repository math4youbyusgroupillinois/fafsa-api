Federal Student Aid (FAFSA) API Docs

This is API documentation for the FAFSA API, allowing you to interact with the API while you learn about all the endpoints available.

You will need an application id and application key before you can use, obtain these for signing up for an API Evangelist API, which is free.

The root URL for these API resources is: http://fafsa.api.publicprivatesector.org.
